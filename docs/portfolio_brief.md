# Portfolio Brief

LabFlow AI Studio is a synthetic AI/LIMS workflow development studio built to demonstrate how I would approach a regulated-domain AI platform problem: deterministic workflow logic first, retrieval with citations second, and model-driven assistance only inside guarded boundaries.

## What It Shows

- lab-domain modeling with units, wells, containers, sample ancestry, quantification, normalization, re-quantification, and downstream QC provenance;
- RAG over synthetic SOP and policy docs with citation discipline;
- eval harnesses for retrieval, answer quality, tool correctness, and corpus drift;
- controlled tools that keep deterministic validation in charge;
- VS Code and FastAPI surfaces for developer workflow;
- AWS-shaped infrastructure and explicit production gaps.

## What It Does Not Claim

It is not production software, a clinical or diagnostic system, a robot controller, a STARLIMS clone, or a reproduction of proprietary SOPs.
